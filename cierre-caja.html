Sistema de Cierre de Caja (Simple)

Control profesional de efectivo y reportes diarios - Sin caché

📖 Guía de Uso del Sistema

🎯 Proceso de 3 Pasos

  • Paso 1: Establecer monto base para el próximo día
  • Paso 2: Contar efectivo real del día actual
  • Paso 3: Registrar gastos, transferencias y generar reporte

💡 Consejos Importantes

  • Los campos se llenan automáticamente con "0" al completar cada paso
  • Use "Exportar Borrador" para guardar progreso sin validación
  • Recaudos, Promociones y Ventas de Cajas se registran como "Negativo" en Entrada Manual
  • Seleccione el punto de venta antes de exportar el reporte final

⚠️ Validaciones

  • El reporte final requiere completar todos los pasos
  • La diferencia máxima permitida es 0.05%
  • Diferencias mayores requieren autorización de administrador
  • Para solicitar autorización escriba en el chat: Administradores o al Responsable

🔐 Nota de Seguridad: Este sistema no guarda datos automáticamente. Use "Exportar Borrador" para preservar su trabajo.
